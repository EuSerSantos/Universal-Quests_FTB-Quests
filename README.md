# 🏆 Universal Quests FTB Quests Pack

A curated collection of ready-to-use FTB Quests for multiple Minecraft mods. Just drag, drop, and play!

---

### 🛠️ Core Dependency
This project relies entirely on the official **FTB Quests** mod. If you are a modpack creator, make sure you have it installed:
* 🌐 **CurseForge:** [FTB Quests (Fabric/Forge)](https://www.curseforge.com/minecraft/mc-mods/ftb-quests-fabric)
* 🐙 **Official Team GitHub:** [FTB Team / FTB Quests](https://github.com/FTBTeam/FTB-Quests)

### 🚀 Contributing (Let's Build Together!)
Want to see your favorite mod covered here? Or maybe you found a typo or a broken reward in one of the current chapters? **Contributions are highly welcome!**

If you want to help:
1. **Fork** this repository.
2. Add your new `.snbt` quest chapters inside the `chapters/` folder (or fix an existing one).
3. Update the `MODS.md` file table with the new mod information.
4. Submit a **Pull Request**! 

*Let's create the biggest central library of pre-made FTB Quests together!*
